# Eigenwise Toolshed docs

This is the [Astro Starlight](https://starlight.astro.build/) documentation site for Eigenwise Toolshed.

```text
npm ci
npm run dev
```

`npm run build` type-checks and builds the static site into `dist/`. GitHub Pages deploys that directory at <https://eigenwise.github.io/eigenwise-toolshed/> after a push to `main`.
