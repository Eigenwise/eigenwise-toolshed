title: Workbench setup
description: Set up and maintain a Claude Code workspace with Workbench.
---

Workbench is the one user-scope install for workspace setup, Toolshed updates, health checks, and retrospectives. Install it, reload Claude Code, then ask it to set up the current project when you are ready.

```text
/plugin install workbench@eigenwise-toolshed --scope user
```

The full setup flow, its choices, and the files it writes are documented here next.
