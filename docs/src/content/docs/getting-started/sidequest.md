Sidequest is a local Kanban board for Claude Code work. Install it when you want a quick thought, bug, or follow-up to become a ticket instead of interrupting the work already underway.

```text
/plugin install sidequest@eigenwise-toolshed
```

This guide will cover board setup, ticket routing, and the dashboard flow in more detail.
